# Pocket Calc

## What is this?

This repository contains code for a simple calculator that I built with jQuery. It's stylized with HTML/CSS and Bootstrap.

## Functionality

The calculator includes operations for the following:
  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Decimal values
